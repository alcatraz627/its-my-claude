- Neon's `darkClass: "light"` is an inverted convention — neon is inherently dark, so "light" is the non-default modifier. Safe to remove since floating toolbar also uses `html.light` class.

- The Edit tool rejects edits on files not read in the current conversation context. When editing in parallel, must read all files first.
- `generate-html.ts --style <name> <output>` requires a full file path for `<output>`, not a directory.
